# 🏥 Nursing Data Center (NDC) - โรงพยาบาลชลบุรี

**ศูนย์ข้อมูลกลุ่มภารกิจการพยาบาล โรงพยาบาลชลบุรี** Web Application ประเภท Web Portal ที่รวบรวมระบบงานและสารสนเทศทางการพยาบาลทั้งหมดไว้ในที่เดียว (Single Point of Access) เพื่ออำนวยความสะดวกให้แก่บุคลากรทางการพยาบาลในการเข้าถึงข้อมูลและเครื่องมือปฏิบัติงานได้อย่างรวดเร็วและมีประสิทธิภาพ

---

## 📑 ระบบที่รวบรวมไว้ (Integrated Systems)

Portal นี้รวบรวมลิงก์ไปยัง Web Application ย่อยจำนวน 6 ระบบ ดังนี้:

1. **ระบบรายงานยอดออนไลน์** (`/reportnso/`)
   - ระบบรายงานสถานะและยอดผู้ป่วยประจำวัน สำหรับการบริหารจัดการเตียง
2. **IC Control Chart** (`/controlchart/`)
   - ระบบกราฟควบคุมงานป้องกันและควบคุมการติดเชื้อในโรงพยาบาล (IC)
3. **รายงานตัวชี้วัด (KPI)** (`/kpi/`)
   - แดชบอร์ดสรุปผลการดำเนินงานตามตัวชี้วัดของกลุ่มภารกิจการพยาบาล
4. **ระบบบริหารผู้ป่วยใน** (`/enn/ward.html`)
   - ระบบจัดการข้อมูลและบริหารจัดการผู้ป่วยสำหรับหอผู้ป่วยใน (Ward)
5. **ระบบผลงานวิชาการ** (`/nsoacademic/`)
   - ศูนย์รวมข้อมูลงานวิจัย นวัตกรรม และผลงานวิชาการของบุคลากรพยาบาล
6. **โปรแกรมคำนวณอัตรากำลังพยาบาล** (`/nwfc/`)
   - เครื่องมือคำนวณอัตรากำลังพยาบาล (Nursing Workforce) ให้เหมาะสมกับภาระงาน

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

- **HTML5**: โครงสร้างหน้าเว็บ
- **Tailwind CSS (via CDN)**: สำหรับการออกแบบ UI ที่เป็น Responsive (รองรับการใช้งานบนมือถือ แท็บเล็ต และคอมพิวเตอร์)
- **FontAwesome**: ไอคอนสัญลักษณ์ต่างๆ เพื่อให้ UI ดูเข้าใจง่ายและสวยงาม
- **Google Fonts (Sarabun)**: ฟอนต์ภาษาไทยมาตรฐานสารบรรณ

---

## 🚀 การติดตั้งและการใช้งาน (Installation & Usage)

เนื่องจากโปรเจกต์นี้เขียนด้วย HTML และ CSS แบบ Native สามารถนำไปใช้งานได้ทันทีโดยไม่ต้องผ่านกระบวนการ Build:

1. คัดลอกไฟล์ `index.html`
2. นำไปวางใน Web Server ของโรงพยาบาล หรือโฮสต์บน GitHub Pages
3. สามารถเปิดไฟล์ `index.html` ผ่านเว็บบราวเซอร์เพื่อใช้งานหน้า Portal ได้ทันที
